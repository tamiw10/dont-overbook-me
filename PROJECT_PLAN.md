# Don't Overbook Me

An AI-powered weekly schedule planning app that helps users balance work, internships, studying, social life, and rest.

## MVP Goal

The first version will not use real APIs yet. It will use mock data to display a weekly planning dashboard.

## MVP Features

- Weekly dashboard
- Mock Google Calendar events
- User preferences form
- Weekly priorities input
- AI-style schedule recommendation display
- Shift availability summary

## Later Features

- Google Calendar integration
- OpenAI API integration
- Supabase database
- Mood and energy tracking
- Calendar event creation
